# Rent-o-Car Frontend

## Installation

### 1. Clone the repository
```sh
git clone https://github.com/Hacker-Man07/Rent-o-Car-Frontend
cd rent-car-frontend
```

### 2. Install dependencies
```sh
yarn
```

## Usage

### Start the application
```sh
yarn start
```
